For each index.html file placed in the src/app/ directory, SKY UX will automatically register a route and create the corresponding component. We also support routes that contain a Route Param.

To automatically create a route param, prefix the folder name with an underscore. The string after the underscore is used as the variable name. For example, if you created a file at src/app/dogs/_id/index.html with the following content:

You are viewing dog {{ id }}.

Visiting the route dogs/42 would set the id variable to "42", which would render as:

You are viewing dog 42.
