Similiar to how you add entry components, app-extras.module.ts is used for module-level providers and services.

Below is an example app-extras.module.ts that imports the DragulaModule.

Please note that DragulaModule is added to both the imports and exports properties of AppExtrasModule as app-extras.module.ts is not the application's root module. import { NgModule } from '@angular/core'; import { DragulaService, DragulaModule } from 'ng2-dragula/ng2-dragula'; // Specify entry components, module-level providers, etc. here. @NgModule({ imports: [ DragulaModule ], exports: [ DragulaModule ], providers: [ DragulaService ], entryComponents: [] }) export class AppExtrasModule { }
